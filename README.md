# 📖 TDD Solver <a name="about-project"></a>

**TDD Solver** is a Ruby project that solve the factorial, reverse and fizzbuzz methods and add tests using the TDD methodology.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need an Ubuntu console (WSL)

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/ivangonzalez224/TDD-Solver.git
```

### Install

Install this project with:

```sh
  cd my-project
  bundle install
```

### Run tests

To run tests, run the following command:

```sh
  cd project
  rspec spec
```
